Сайт подходит для разрешений 1440x900 и 1368x768
В конфигурации php в easyphp установить session.auto_start = 1
Для входа в часть преподавателя необходимо добавить к адресу сайта /admin.php
После чего выбрать регистрацию и пройдя регистрацию, в базе данных установить
аккаунту LevelAcess = 1, после чего войти.
Медиа-менеджер: медиа-менджер позволяет удобно хранить файлы на сервере,
есть возможность создавать,удалять папки/изображения. Для того чтобы удалить
сразу несколько изображений можно выделить их правой кнопкой мышки и нажать удалить.
Чтобы взять локальный адрес картинки необходимо нажать на неё левой кнопкой мышки,
после чего скопировать адрес в адресной строке.
О функциях учебниках можно подробней узнать в файле Отчет.docx
Первый проект на PhP был сделан в колледже.
